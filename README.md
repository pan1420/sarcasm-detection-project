# Sarcasm Detection Project

This project uses machine learning to detect sarcasm in text headlines.

## Models Used
- Support Vector Machine (SVM)
- Logistic Regression

## Features
- Headline length
- Exclamation & question marks
- Sentiment score using VADER

## Accuracy
- SVM: 71.33%
- Logistic Regression: 70.33%

## How to Run
1. Make sure Python is installed.
2. Install required libraries:
   pip install -r requirements.txt
3. Run the project:
   python main.py
